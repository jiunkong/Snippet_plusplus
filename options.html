Syntax Guide

Basic Placeholders

{{#0}}, {{#1}}, ... / {{cursor}} 텍스트 삽입 후 Tab 키를 눌러 이동할 커서 순서를 지정합니다.
Item: {{#0}}
{{# expr }} 표현식을 사용하여 커서 순서를 동적으로 지정합니다.
{{# $var + 1 }}

Info System Functions

{{date()}}, {{time()}}, {{datetime()}} 현재 날짜와 시간을 삽입합니다.
{{date('YYYY/MM/DD (ddd)')}}
{{time('HH시 mm분')}}
{{url()}}, {{domain()}}, {{title()}} 현재 탭의 주소, 도메인, 제목 정보를 삽입합니다.
{{clipboard()}} 클립보드에 복사된 텍스트를 비동기로 가져와 삽입합니다.

Input Interaction

{{prompt('Message')}} 사용자로부터 입력을 받는 팝업을 띄우고, 입력된 값을 본문에 삽입합니다.
Name: {{prompt('이름을 입력하세요')}}
{{choice('Title', 'Opt1', 'Opt2', ...)}} 여러 옵션 중 하나를 선택하는 메뉴를 띄웁니다. 숫자키(1~9)로도 선택 가능합니다.
Food: {{choice('메뉴 선택', '치킨', '피자', '햄버거')}}
💡 Tip: 동적 프롬프트
프롬프트 메시지에도 변수나 수식을 사용할 수 있습니다.
예: {{prompt($title + '의 가격')}}

Random Generators

{{uuid()}} 무작위 UUID v4 식별자를 생성하여 삽입합니다.
{{random(a ~ b)}} 지정한 범위(a와 b 사이)의 무작위 정수를 생성합니다.
{{random(1 ~ 100)}} / {{random($min ~ $min + 10)}}
{{random(expr1, expr2, ...)}} 콤마로 구분된 인자 중 하나를 무작위 선택합니다.
{{random('치킨', '피자', 10+20)}}

String Manipulation

{{len(s)}} 문자열의 길이를 반환합니다.
{{upper(s)}}, {{lower(s)}}, {{capitalize(s)}} 대문자, 소문자, 첫 글자 대문자 변환을 수행합니다.
{{substr(s, start, len)}} 지정한 위치부터 원하는 길이만큼 문자열을 자릅니다.
{{substr('Snippet++', 0, 7)}} → Snippet
{{replace(s, old, new)}} 문자열 내의 특정 텍스트를 다른 텍스트로 치환합니다.
{{trim(s)}} 문자열 앞뒤의 모든 공백을 제거합니다.

Type Conversion

{{str(val)}} 값을 문자열로 강제 변환합니다.
{{num(val)}} 값을 숫자형(소수점 포함)으로 변환합니다.
{{int(val)}} 값을 정수형으로 변환합니다. (소수점 절삭)
{{int('10.5') + 1}} → 11

Math Functions

{{round(n)}}, {{ceil(n)}}, {{floor(n)}} 반올림, 올림, 내림을 수행합니다.
{{min(a, b, ...)}}, {{max(a, b, ...)}} 주어진 값들 중 최솟값 또는 최댓값을 반환합니다.
{{max(1, 10, $var)}}

Arguments Trigger Capture

${name:regex:flags} 트리거에서 변수를 캡처합니다. (플래그 지정 가능)
;;${id:\d+} (결과: ;;123)
{{$name}} 캡처한 변수를 본문에서 사용합니다. (하늘색 강조)

Logic Conditionals

{{if expr}} ... {{endif}} 조건에 따라 텍스트를 노출합니다.
{{if $type == 'bug'}}[BUG]{{else}}[TASK]{{endif}}
{{elif expr}}, {{else}} 다중 조건이나 예외 처리를 지원합니다.

Logic Switch/Case

{{switch expr}} ... {{endswitch}} 하나의 식 결과를 여러 값과 비교합니다.
{{case val}} ... {{endcase}} 비교 대상 블록입니다. 콤마로 여러 값을 지정할 수 있습니다.
{{case 'A', 'B'}}

Loops Iteration

{{for expr}} ... {{endfor}} 내용을 지정된 횟수만큼 반복합니다.
{{index}} 현재 인덱스를 표시합니다. (0부터 시작)
{{for 3}} ({{index}}) {{endfor}}

Expr Variables & Math

{{set $var = expr}} 변수를 선언하고 계산 결과를 저장합니다. +=, -=, *=, /=, %= 연산자로 기존 변수 값을 수정할 수 있습니다.
{{set $count = 0}}
{{for 5}} {{set $count += 1}} {{endfor}}
Result: {{$count}}
{{expr}} (Math & Logic) 중괄호 안에서 연산(+, -, *, /, %)과 함수 호출을 수행합니다.
Score: {{($p1 + $p2) / 2}}
💡 Tip: 변수 유효 범위 (Scope)
외부에서 선언된 변수는 if, for, switch 블록 내부에서도 수정할 수 있습니다. 단, 블록 내부에서 처음 정의된 변수는 블록이 종료되면 자동으로 사라집니다.

Misc Formatting

{{backspace()}}, {{bs()}} 백스페이스 동작을 수행하여 이전 문자를 하나 지웁니다.
ABC{{bs()}}D → Result: ABD
\ (Line Continuation) 줄 끝에 \를 붙이면 다음 줄과의 개행 및 다음 줄의 들여쓰기를 제거합니다.
